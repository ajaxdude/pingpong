🔧 Setup Required

Pingpong needs llama.cpp to be running

Connection Error: Cannot connect to {{endpoint}}. Please ensure llama.cpp is running.

1 Install llama.cpp

If you haven't installed llama.cpp yet, you can install it with:

git clone https://github.com/ggerganov/llama.cpp.git cd llama.cpp make

2 Download a Model

Download a compatible model (e.g., Llama 3, Mistral, etc.):

# Example: Download Llama 3 8B Instruct wget https://huggingface.co/meta-llama/Meta-Llama-3-8B-Instruct-GGUF/resolve/main/meta-llama-3-8b-instruct.q4_k_m.gguf

3 Start llama.cpp Server

Start the llama.cpp server on port 8080:

# Replace with your actual model file path ./llama-server -m path/to/your/model.gguf -p 8080
Tip: You can run llama.cpp on a different port by updating the endpoint in pingpong.config.json

4 Verify Connection

Test that llama.cpp is running correctly:

curl http://127.0.0.1:8080/v1/models
